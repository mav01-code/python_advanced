Meta characters to skip
. ^ $ * { } [ ] ( ) | \
-----------------------------------------------------------------
* : 0 or more
+ : 1 or more
? : 0 or 1
{n} : exactly n numbers
{mi,mx} : range of numbers. min-max
-----------------------------------------------------------------
. : any character except new line
^ : starts with
$ : ends with
\d : Digit (0-9)
\D : not a Digit
\w : word characters (a-z)(A-Z)
\W : not a word character
------------------------------------------------------------------
